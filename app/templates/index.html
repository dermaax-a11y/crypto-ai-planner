Seriöser Krypto-Planer · lokale Demo-App

Crypto AI Planner

Diese App erstellt einen datenbasierten Investmentplan für Krypto, statt unrealistische Renditen zu versprechen. Die Entscheidungen sind nachvollziehbar: Marktregime, Liquidität, Momentum, Qualität und Risikoprofil fließen sichtbar in das Ergebnis ein.

App-Status

Version 1.0 · lokal lauffähig · API inklusive
  • Web-Oberfläche
  • REST API
  • What-if Analyse
  • Docker-Setup

Nutzerprofil

Das Modell ist absichtlich konservativ aufgebaut: Qualitätsfilter zuerst, Risiko danach, Hype zuletzt.

So startest du die App

uvicorn app.main:app --reload

Danach im Browser: http://127.0.0.1:8000

docker compose up --build

Alternativ komplett per Docker.
